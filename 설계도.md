## 1. 페이지 구성 (총 7개)

| # | 페이지 | 핵심 요소 |
|---|--------|----------|
| 1 | **Home** | 풀블리드 히어로 + 거대 세리프 워드마크, Bestsellers 캐러셀, 쇼퍼블 이미지(핫스팟), 카테고리 텍스트 클라우드, Bedroom/Dining 캐러셀, About(다크 아크 전환), 인스타 콜라주, 풋터 |
| 2 | **Catalog** | 필터 칩 + Sort, 3열 상품 그리드, 중간 삽입 쇼퍼블 이미지, Load more + 페이지네이션 |
| 3 | **Product Card** (Shibari) | 갤러리 + 썸네일, About 섹션, 스펙 다운로드 링크, 라이프스타일 영상, You May Also Like, 디자이너(Studiopepe) 소개, Our Journal |
| 4 | **Checkout** | 다크 포토 배경 위 화이트 모달 — Contact / Delivery / Payment + Your order 요약 |
| 5 | **Account** | 로그인(이메일+소셜), 대시보드(Review/Orders/Profile/Addresses/Wishlist), Wishlist 그리드 |
| 6 | **404** | 거대 워드마크 겹침 + "erreur 404" 이탤릭 포토 카드 |
| 7 | **Search** | 풀와이드 오버레이 — 인풋 + 결과 그리드 + "62 products found" |

**오버레이 컴포넌트**: 메가 메뉴(카테고리 리스트 + 이미지 4타일), 카트 드로어(Your order), 검색 오버레이

## 2. 디자인 시스템

### 컬러 (토큰)
```css
--bg-ivory: #F4F2ED;      /* 베이스 */
--bg-dark: #1E1D1B;       /* 다크 섹션 (145deg 사선 그라디언트 질감) */
--text-primary: #1A1A1A;
--text-inverse: #F4F2ED;
--line: rgba(26,26,26,.18);
```
포인트 컬러 없음 — **사진의 웜톤이 유일한 컬러**. UI는 완전 모노크롬.

### 타이포그래피
| 역할 | 폰트 | 특징 |
|------|------|------|
| Display/헤딩 | **Cormorant Garamond** | 시그니처: 한 단어 안 로만+이탤릭 믹스 → Best*sellers*, Bed*room*, Pto*ducts* |
| UI/본문 | **Helvetica Neue** (대체: Inter/Neue Haas) | 소문자 라벨은 uppercase + letter-spacing |
| 워드마크 | Cormorant 대문자 초대형 | vw 단위, 이미지와 겹침(z-index 레이어링) |

### 핵심 컴포넌트
- **상품 카드**: 옅은 회백 bg 이미지 / 이름 + 디자이너(회색) + 가격 / ♡ 위시 / "Add to cart" 언더라인 링크
- **아크 섹션 전환**: 아이보리 → 다크로 넘어갈 때 거대한 타원 곡선 (border-radius 50% 확장 div)
- **핫스팟**: 사진 위 ⊕ 버튼 → 미니 상품카드 팝오버
- **카테고리 클라우드**: 세리프 대형 텍스트 + 위첨자 카운트(240+), hover 시 썸네일 이미지 삽입
- **READ / SHOP 내비**: 글자 사이에 가구 이미지 삽입 + hover 취소선

## 3. 시그니처 인터랙션 (난이도순)

1. **쉬움** — 필터 칩, 캐러셀(Swiper), 위시 토글, 아코디언
2. **중간** — 메가 메뉴/검색/카트 오버레이, 핫스팟 팝오버, 카테고리 hover 이미지 스왑
3. **중상** — 아크 곡선 스크롤 전환(GSAP ScrollTrigger), 워드마크 패럴랙스 겹침, 히어로 텍스트 리빌
4. **디테일** — Lenis 스무스 스크롤, 이미지 hover scale(1.05, ease 0.8s), 페이지 전환 페이드

(포트폴리오용 정적 사이트 — 프레임워크 불필요, 배포는 Vercel/Netlify)


## 5. 제작 순서 (5 Phase)

| Phase | 내용 | 산출물 |
|-------|------|--------|
| **P1** | 토큰 + 베이스 + 공통 컴포넌트(nav, footer, 상품카드) | 디자인 시스템 완성 |
| **P2** | **Home 풀빌드** — 이 사이트의 80% 난이도가 여기 집중 | 히어로~풋터 + 아크 전환 |
| **P3** | Catalog + Product Card | 그리드·필터·갤러리 |
| **P4** | 오버레이 3종(메뉴/검색/카트) + Checkout + Account + 404 | 전 페이지 연결 |
| **P5** | 반응형(1024/390) + 인터랙션 폴리시 + 성능 최적화 | 배포 |

각 Phase 끝날 때마다 Git 커밋. P2가 승부처 — Home만 완성돼도 포트폴리오 스크린샷 가능.

## 6. 에셋 전략 (중요)

- 레퍼런스 이미지는 **Visionnaire 브랜드 소유** → 공개 포트폴리오에 그대로 쓰면 리스크
- **권장**: 나노바나나로 동일 무드 생성 — "warm beige luxury Italian furniture, boucle armchair, travertine villa, golden hour" 톤
- 필요 컷: 히어로(빌라+소파) 2, 상품 누끼 ~15, 라이프스타일 6, 인물(디자이너) 1
- 사이트명도 변경 권장 (예: **AURELIA** 확장 or 신규 네이밍) → "리디자인 컨셉" 명시로 저작권 클린
